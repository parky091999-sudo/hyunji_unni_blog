# 현지언니 블로그 수동 설정 가이드

> 자동화로 처리되지 않는 항목들. 나중에 여유 있을 때 순서대로 진행.

---

## 1. 스킨/테마 선택 ⭐ 중요

**경로**: 블로그 관리 → 꾸미기 설정 → 스킨 선택

**추천 스킨 기준** (살림/신혼 테마):
- 밝고 깨끗한 흰색 배경
- 미니멀하고 여성스러운 느낌
- 폰트가 읽기 편한 것
- **추천 계열**: "Simple", "Clean", "Modern" 계열

**설정 방법**:
1. `blog.naver.com → 내 블로그 → 관리 → 꾸미기 설정 → 레이아웃·위젯 설정`
2. 스킨 선택 → 미리보기로 확인 → 적용

---

## 2. 레이아웃 설정

**경로**: 블로그 관리 → 꾸미기 설정 → 레이아웃·위젯 설정

**권장 설정**:
- 전체 정렬: **가운데 정렬** (왼쪽 정렬보다 가독성 좋음)
- 레이아웃: **1열** (사이드바 없는 넓은 본문) 또는 **오른쪽 사이드바**
- 위젯 추가 권장: 카테고리, 최근 댓글, 이웃 커넥트

---

## 3. 프로필 사진 업로드 ⭐ 중요

**경로**: 블로그 관리 → 기본 설정 → 프로필

**권장 사항**:
- 밝고 친근한 느낌의 일러스트 또는 실제 사진
- 정사각형 (1:1 비율), 최소 200×200px
- 배경이 깔끔한 것
- 20대 여성, 주부, 따뜻한 분위기

---

## 4. 글씨체/에디터 기본값 설정 (자동화 안 된 경우)

**경로**: 블로그 관리 → 기본 설정 → 글쓰기 기본 환경

**권장 설정**:
- 기본 서체: **나눔스퀘어** (가독성 최고)
- 글자 크기: **16pt** (모바일 최적)
- 줄 간격: **180%** (답답하지 않게)
- 에디터: **스마트에디터 ONE** (자동화 호환)

---

## 5. 메뉴 설정 (상단 메뉴바)

**경로**: 블로그 관리 → 메뉴·글·동영상 관리 → 메뉴 설정

**권장 설정**:
- 지도: **OFF** (불필요)
- 서재: **OFF** (불필요)
- 메모: **OFF** (불필요)
- 태그: **ON**
- 안부게시판: **이웃만 쓸 수 있게** (스팸 방지)

---

## 6. 글 보기 설정

**경로**: 블로그 관리 → 메뉴·글·동영상 관리 → 카테고리 관리

**권장 설정**:
- 페이지당 글 수: **1개** (스크롤 집중도 향상)
- 글 보기 방식: **앨범형** (썸네일 보이기) 또는 **목록형**

---

## 7. 이웃 커넥트 / 서치어드바이저 등록

**서치어드바이저 등록** (상위노출에 도움):
1. `searchadvisor.naver.com` 접속
2. 사이트 소유 확인 (RSS 또는 메타태그 방식)
3. 블로그 RSS URL: `https://rss.blog.naver.com/hyunji_unni.xml`

---

## 8. 카테고리 구성 (자동화 실패 시 수동)

**경로**: 블로그 관리 → 메뉴·글·동영상 관리 → 카테고리 관리

```
살림꿀팁
  ├── 청소&정리
  ├── 요리&식비절약
  ├── 인테리어&수납
  ├── 절약&재테크
  └── 쇼핑정보
신혼일상
```

**설정 방법**:
1. 카테고리 추가 버튼 클릭
2. 상위 카테고리 이름 입력 후 Enter
3. 하위 카테고리 추가 버튼 클릭 후 이름 입력
4. 순서 드래그로 조정
5. 확인 버튼

---

> 자동 설정: `python scripts/setup_blog.py` (로컬에서 실행)
> 자동으로 설정되는 것: 카테고리, 블로그명/닉네임/소개글, 에디터 기본 서체
